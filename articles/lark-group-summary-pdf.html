Codex飞书群聊总结操作指南

10分钟阅读
准备什么 下载飞书 注册并登录 创建或加入群 准备skill 完成授权 总结群聊 查看结果 常见问题

准备什么

开始之前，你需要准备好这几样东西：

  • 一个飞书账号
  • 一个飞书群
  • 配置完成的 Codex App
  • lark-group-summary-pdf 这份 skill

这套流程第一次配置会稍微长一点。配置好以后，后面基本就是一句话让 Codex 总结。

1. 下载飞书

  1. 打开飞书官网： https://www.feishu.cn/
  2. 点击“下载飞书”。
  3. 按你的电脑系统下载安装包。
  4. 安装完成后，打开飞书。

2. 注册并登录飞书账号

  1. 打开飞书客户端。
  2. 选择手机号、邮箱或扫码登录。
  3. 如果还没有账号，按页面提示注册。
  4. 登录成功后，确认自己能正常进入飞书主界面。

如果公司已经有飞书组织，优先用组织给你的方式登录。这样后面加入群、授权应用会少很多麻烦。

3. 创建或加入一个飞书群

你可以自己建一个测试群，也可以加入已有群。

创建测试群

  1. 打开飞书。
  2. 点击左侧“消息”。
  3. 点击右上角的加号。
  4. 选择“创建群聊”。
  5. 给群起一个名字，比如 test。
  6. 建好以后，在群里发几条消息，最好也发一张图片，方便测试总结效果。

加入已有群

如果别人已经建好了群，让对方把你拉进群，或者给你群二维码/邀请链接。你必须能在自己的飞书里看到这个群，Codex 才能读取你有权限看到的消息。

4. 为Codex准备skill

先下载 lark-group-summary-pdf skill ，然后把压缩包解压到 Codex 的 skills 目录里。

macOS / Linux 安装位置

~/.codex/skills/lark-group-summary-pdf

Windows 安装位置

%USERPROFILE%\.codex\skills\lark-group-summary-pdf

解压完成后，确认这个文件夹里能看到 SKILL.md。然后重启 Codex App，让它重新加载新安装的 skill。

5. 按提示完成飞书授权

第一次使用时，Codex 会要求你授权飞书。

你要做的是：

  1. 打开 Codex 给你的链接。
  2. 用飞书账号登录。
  3. 页面要求输入验证码时，就输入 Codex 给你的验证码。
  4. 页面要求确认授权时，点击确认。
  5. 如果页面让你选择权限，尽量选择全部业务域和全部权限。

授权时不用自己理解每个权限。大致可以理解为：Codex 需要读取你能看到的群消息、下载图片、生成 PDF，并把 PDF 发回群里。

如果某个权限没给成功，Codex 会提示缺什么权限，并再次给你授权链接。照着打开链接确认即可。

6. 让 Codex 总结群聊

配置完成后，你可以直接对 Codex 说：

总结飞书群聊内容，并把PDF发回群里

默认情况下，skill 会总结两个时间范围：

  • 今日内容
  • 近七日内容

Codex 会把两部分消息合并去重，读取文字消息，下载图片资源，必要时查看图片内容，然后生成一份 PDF。

7. 你会在群里看到什么

正常情况下，飞书群里会收到一个 PDF 文件。通常包含：

  • 群聊名称
  • 总结时间范围
  • 关键要点
  • 待办和结论
  • 消息时间线
  • 图片或文件摘要

如果飞书不允许直接发送文件，Codex 会把 PDF 上传到飞书云空间，再把链接发回群里。

8. 常见问题

Codex 找不到群

先确认你自己的飞书账号能看到这个群。如果你不在群里，Codex 也读不到。

你也可以把群名发给 Codex，让它搜索群；如果还是找不到，把群 ID 发给 Codex。

授权一直提示缺权限

按 Codex 给出的链接重新授权。如果某个权限最终还是无法授予，通常是飞书组织或应用后台限制，不是你操作错了。

PDF 没有直接发成文件

有时飞书会限制群文件上传权限。Codex 会自动改用备用方式：上传到飞书云空间，然后把链接发回群里。

图片能不能总结

可以。Codex 会下载群消息里的图片，然后查看图片内容，再把图片摘要写进 PDF。

能不能自动每天总结

这份指南只覆盖手动触发。你可以每天在 Codex 里说“总结今天的飞书群聊并发回群里”。如果要每天固定时间自动总结，需要再配置定时任务。
