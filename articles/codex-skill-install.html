寻找、安装skills的方法

12分钟阅读

目录

Skills是什么 安装在哪里 寻找Skills的方法 让Codex生成Skills

1. Skills是什么

你可以把 Skills 理解成给 Codex 安装的一份“专项工作说明书”。

一个 Skills 通常是一个文件夹，里面最重要的是 SKILL.md。

my-skill/
  SKILL.md
  scripts/
  references/
  assets/
SKILL.md
必须有，用来告诉 Codex 这个 skill 是什么、什么时候用、怎么做。
scripts/
可选，用来放辅助脚本。
references/
可选，用来放参考资料。
assets/
可选，用来放模板、图片、示例文件。

最小可用结构只有一个文件：

my-skill/
  SKILL.md

Skills 通常可以自动触发，也可以输入美元符号 $ 加 skill 名称手动调用，例如： 

$jhss-weekly-ppt 帮我生成周报

2. Skills安装在哪里

macOS / Linux 常用路径

个人全局 skill 放在用户目录下的 .codex/skills。

~/.codex/skills

完整展开后，在 macOS 上通常类似：

/Users/<你的用户名>/.codex/skills

Windows 常用路径

Windows 上对应的是用户目录下的 .codex\skills。

%USERPROFILE%\.codex\skills

完整路径通常类似：

C:\Users\<你的用户名>\.codex\skills

3. 寻找Skills的方法

办法一：用 Codex 内置的 $skill-installer

这是最适合新手的方法。你可以直接在 Codex 里输入：

$skill-installer

也可以直接让它安装某个 skill：

$skill-installer gh-address-comments

OpenAI 官方 openai/skills 仓库说明，可以通过 $skill-installer 安装 curated skills，也可以指定 experimental skill 文件夹。

  • 你不想手动下载。
  • 你不知道 skill 文件该放哪。
  • 你想安装官方或 curated skills。

办法二：去 GitHub 搜索现成 skill

推特、抖音、小红书等社区有时会推荐一些 skills。你通常可以在 GitHub 找到它们。

社区仓库要注意安全，尤其是带 scripts/ 的 skill。

办法三：从 Claude / Agent Skills 生态迁移

很多 skill 采用通用的 Agent Skills 结构，也就是 skill-folder/SKILL.md。 所以有些 Claude Code、OpenClaw、Codex skill 之间可以互相参考。

新手可以先不管兼容细节，只记住：核心是 SKILL.md，路径是 ~/.codex/skills。

4. 让Codex生成Skills

你也可以直接对 Codex 说：

使用 $skill-creator 帮我创建一个 Codex skill，用来阅读 A 股券商研报。
要求面向新手，输出投资逻辑、估值假设、风险点、财务数据、后续追问清单。

如果你的目标还不清楚，先写明使用场景、输入材料、输出格式和安全边界。Codex 能更稳定地把这些要求整理进 SKILL.md。
