Codex配置教程

01

第一步：安装 Codex App

根据当前系统选择对应安装方式。macOS 和 Linux 可从 codex-app-mirror releases 下载最新版安装包，Windows 请通过 Microsoft Store 安装。

  1. 打开 Codex App GitHub Releases，选择标记为 Latest 的版本。
  2. 在 Assets 中下载与系统匹配的安装包。
  3. 运行安装包并按系统提示完成安装。
  4. 安装完成后打开 Codex App，确认应用能正常启动。
  1. 打开 Microsoft Store。
  2. 在 Microsoft Store 中搜索 Codex。
  3. 进入 Codex 页面并点击“获取”或“安装”。

  2. 在 Assets 中下载与系统匹配的安装包。
  3. 运行安装包并按系统提示完成安装。
  4. 安装完成后打开 Codex App，确认应用能正常启动。
02

第二步：安装 Codex++

访问 codex++ releases 下载最新版安装包，并按当前系统完成安装。

  1. 打开 Codex++ GitHub Releases，选择标记为 Latest 的版本。
  2. 在 Assets 中下载与系统和芯片架构匹配的安装包。
  3. 运行安装包并按系统提示完成安装。
  4. 安装完成后重新打开 Codex 或终端，再检查 Codex++ 是否正常加载。

macOS 如果提示无法打开来自互联网的应用，请在“系统设置 → 隐私与安全性”中允许打开对应应用。

03

第三步：注册账号

前往 嘉合杉升AI接入系统 完成注册。注册时用户名请使用真名，方便管理员识别并给予额度。本系统获得的 API Key 只能用于 Codex，不可接入龙虾或其他工具。

  1. 打开注册页面并按提示填写账号信息。
  2. 用户名填写本人真名，避免使用昵称或缩写。
  3. 提交注册后等待管理员审核并分配额度。
  4. 额度生效后，只在 Codex 中使用对应 API Key，不要接入龙虾或其他工具。
04

第四步：添加令牌

  1. 登录嘉合杉升 AI 接入系统，进入控制台。
  2. 在控制台中打开“令牌管理”。
  3. 点击“添加令牌”。
  4. 在“基本信息”中填写“名称”，名称可以随便命名。
  5. 复制创建令牌后生成的密钥，确认它以 sk- 开头。
  6. 将 API Key 保存到本地，后续配置 Codex 时会用到。
05

第五步：配置供应商

确认桌面上已有 Codex、Codex++ 和 Codex++ 管理工具。先关闭后台中的相关应用会话，再打开 Codex++ 管理工具，按右侧表单填写供应商配置。

  1. 确认后台中的 Codex、Codex++ 和 Codex++ 管理工具会话都已关闭。
  2. 打开 Codex++ 管理工具。
  3. 在侧边栏选择“供应商配置”。
  4. 点击“+ 添加供应商”。
  5. 按照右侧表单完成配置，其中 Key 使用第四步保存的内容。
  6. 点击上方“保存”按钮保存配置。

06

第六步：启动配置好的 Codex

  1. 回到“供应商配置”列表。
  2. 选择并使用刚刚添加的供应商配置。
  3. 点击上方“重启 Codex++”按钮，让配置生效。
  4. 双击桌面上的 Codex++（注意不是Codex），启动已经配置好的 Codex。
