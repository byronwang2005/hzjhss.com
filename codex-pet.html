Codex宠物的用法

5分钟阅读

核心意义 如何开启宠物 如何创造独一无二的宠物

1. 核心意义：不只是卖萌的“情绪价值”

Codex宠物是OpenAI在2026年推出的一项趣味功能。它是一个为你实时报告Codex工作进度的可爱小伙伴，充当着可视化的“灵动岛”角色。

Codex宠物主要显示三种工作状态：

  • 任务执行中（Running）：宠物会显示动态的忙碌动画，例如挠头、翻箱倒柜，让你知道它正在后台处理任务。
  • 等待输入（Waiting for Input）：宠物会安静地待机，提醒你需要提供下一步指令才能继续。
  • 任务完成（Ready for Review）：宠物会通过挥手、闪烁等动作发出轻微的视觉提示，提醒你回来验收工作成果。

2. 如何开启宠物？

  1. 确保基础环境：这是桌面端的专属功能，需要先下载并安装 Codex 的 Windows 或 macOS 桌面应用。
  2. 简单召唤：在 Codex 的应用内输入框中，直接键入 /pet 并回车，宠物就会出现在屏幕上。
  3. 快捷设置：通过快捷键 Cmd + K（Mac）或 Ctrl + K（Windows/Linux）调出命令菜单，搜索“Wake Pet”来唤醒，或搜索“Tuck Away Pet”来收起宠物。你也可以进入 Settings → Appearance → Pets，在内置的 8 款像素风宠物中选择并切换。

3. 如何创造独一无二的宠物？

  1. 安装孵化技能：先安装 hatch-pet 这个技能，然后在命令菜单中执行 “Force Reload Skills” 来刷新加载。
  2. 宠物孵化：使用 /hatch 命令，Codex 会根据你最近的代码或你提供的图片描述，为你生成一个定制宠物。
  3. 获取社区宠物：访问 Petdex社区画廊平台等社区平台 ，上面已有数百款玩家创作的开源宠物可供免费下载安装。
